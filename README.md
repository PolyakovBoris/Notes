# консольная записная книжка
main находится в app.py

## основное меню
1. - save note
2. - find and read note
3. - edit note
0. - exit

## возможности
* сохранение ID, названия и тела заметки с отметкой даты

* доступен поиск по ID, названию, тексту, дате

* корректировка заметок по ID

* удаление по ID
